# Sentinel Project (RPG Game)

Этот проект представляет собой Action-RPG, разрабатываемую на игровом движке **Godot 4**.

## 🏗 Стек технологий

### Игровой движок

- **Godot 4.6** (Mobile renderer)
- **Язык:** GDScript

### Архитектура

- **Компонентная система:** Использование композиции (HealthComponent, HitboxComponent) вместо глубокого наследования.
- **Event Bus:** Глобальная шина событий (`Events.gd`) для развязки логики.
- **Dependency Injection:** Автоматический поиск зависимостей при старте сцены.

### Инструменты качества (Quality Assurance)

- **Тестирование:** [GUT (Godot Unit Test)](https://github.com/bitwes/Gut) — интеграционные и юнит-тесты.
- **Линтинг:** `gdlint` (через gdtoolkit).
- **Форматирование:** `gdformat` (стиль кода, близкий к Black).
- **Автоматизация:** `uv` (управление зависимостями) + PowerShell Core.

## 🚀 Быстрый старт (Запуск игры)

### Через редактор Godot

1.  Откройте Godot Project Manager.
2.  Нажмите **Import**.
3.  Выберите файл `project.godot` в корневой папке.
4.  Нажмите кнопку Play (F5).

### Через VS Code

Если вы хотите редактировать код, запускать тесты и использовать автодополнение:

👉 **[Перейти к руководству разработчика (docs/DEVELOPMENT.md)](docs/DEVELOPMENT.md)**

В руководстве описано:

- Как настроить Python-окружение одной кнопкой.
- Как настроить VS Code (скрытие лишних файлов, пути).
- Как запускать авто-форматирование и тесты.
